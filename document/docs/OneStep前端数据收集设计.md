# OneStep前端数据收集设计

## 页面留存时间统计（数据独立） 

基于vue-router的beforeEach做，统计每个页面的停留时间，判断用户功能关注重点。

### 计算方式

用户每次访问网站，统计各个页面停留时间所占比例。

### 数据存储

每个用户建立专门的log日志，命名为：userid_time.log
