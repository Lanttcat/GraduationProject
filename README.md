# GraduationProject毕业设计相关

## 环境依赖
node 7.6+
python3+

## 部署步骤
1. git clone

2. 切换到onestep目录执行npm install  //安装node运行环境

3. 开发环境：npm run dev   //前端调试：127.0.0.1:3000


## 目录结构描述

├── document                    // 相关技术文档总结
